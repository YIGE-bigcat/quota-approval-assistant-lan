# 审批同步助手 Android 版

这是手机端原生安装包工程。它不再依赖浏览器通知，而是使用 Android 原生前台服务轮询电脑端 `codex-watch-bridge`，发现待审批请求后发系统通知。

## 功能

- 手机 App 内嵌现有 Dashboard，可查看 5 小时额度、本周额度、待审批和最近审批。
- 后台监听服务每 8 秒请求一次电脑端 `/api/status`。
- 新待审批出现时发送系统通知。
- 通知包含任务标题、审批原因、5 小时额度、本周额度。
- 通知动作包含“批准”和“拒绝”。
- 手机上的通知可通过华为运动健康同步到 GT5；手表是否显示动作按钮取决于手表固件和运动健康对通知动作的兼容性。

## 手机使用

1. 电脑端先启动 `审批同步助手`，保证手机能访问电脑局域网地址，例如 `http://192.168.1.100:8788`。
2. 安装 APK。
3. 打开 App，填写电脑 Bridge 地址和当前桥接服务 token。
4. 点“保存并提醒”。
5. 系统弹出通知权限时选择允许。
6. 点“后台权限”，把本应用加入允许后台运行/忽略电池优化。

## 手表使用

1. 手机和 GT5 先在华为运动健康里保持正常连接。
2. 华为运动健康里打开“设备 > GT5 > 消息通知”。
3. 在应用列表中允许“审批同步助手”的通知同步。
4. 触发一次 Codex 权限审批。
5. 手表应收到审批通知摘要。
6. 如果手表显示“批准/拒绝”按钮，可以直接在表上操作；如果不显示，点击通知后在手机 App 或电脑网页端审批。

第一版不依赖原生手表 App。GT5 原生 App 需要单独走 HarmonyOS/运动健康可安装能力验证。
